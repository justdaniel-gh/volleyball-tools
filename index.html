This tool attempts to find the scores in the last match in order for one specific team to be the winner in a three-way tie.

This is very much still a work in progress, and only supports 3 teams, with 3 matches, with at least 2 matches filled.

Also, only solves for team 3 to win. Will fix soon...

Usage:

    1. Enter scores for game 1 and game 2 (only sets 1+2 are supported ATT)
    2. Press Calculate
